# Data Structures and Algorithms — CSE222

A collection of homework assignments completed for the CSE222 Data Structures and Algorithms course at Gebze Technical University. Each subdirectory is a self-contained assignment with its own README, build instructions, and source code.

## Assignments

| # | Name | Topic | Language |
|---|------|-------|----------|
| 1 | [Hardware Abstraction Layer Simulation](./hw1-hardware-abstraction-layer) | OOP design, polymorphism, interface-based abstraction | Java |
| 3 | [Queue-Based Command Processing & Stack-Based Operation Logging](./hw3-queue-stack-logging) | Queue (FIFO command buffering), Stack (LIFO operation logging) | Java |
| 4 | [Deep Space Planetary System Analysis](./hw4-deep-space-planetary-system) | General (N-ary) tree, recursion, Stack-based path reconstruction | Java |
| 5 | [Task Scheduling Using a Min Heap](./hw5-min-heap-task-scheduler) | Binary min heap (from scratch), Priority Queue, generics | Java |
| 6 | [High-Performance Spell Checker with a Custom HashMap and Set](./hw6-custom-hashmap-spellchecker) | Hash table from scratch (open addressing, linear probing, tombstones, rehashing), edit distance | Java |
| 7-8 | [Sorting Algorithms, Adjacency Matrix, and Graph Coloring](./hw7-8-sorting-graph-coloring) | Quicksort/insertion/selection sort, adjacency matrix graph, custom Collection implementation, greedy graph coloring | Java |

HW2 was a pen-and-paper mathematical assignment and is not included here. HW7-8 was the final assignment of the course.

## Notes

- Each assignment folder includes a `README.md` describing the problem, the design approach, and how to build and run the code.
- Grading rubrics and course materials beyond the problem statements are excluded. Where an assignment depended on instructor-provided scaffolding (interfaces, abstract classes, a driver `Main`, or an algorithm the student wasn't required to implement) to compile and run, that scaffolding is included for completeness — each assignment's README states exactly which parts were implemented by the student and which were provided.
