# Data Structures and Algorithms — CSE222

A collection of homework assignments completed for the CSE222 Data Structures and Algorithms course at Gebze Technical University. Each subdirectory is a self-contained assignment with its own README, build instructions, and source code.

## Assignments

| # | Name | Topic | Language |
|---|------|-------|----------|
| 1 | [Hardware Abstraction Layer Simulation](./hw1-hardware-abstraction-layer) | OOP design, polymorphism, interface-based abstraction | Java |
| 3 | [Queue-Based Command Processing & Stack-Based Operation Logging](./hw3-queue-stack-logging) | Queue (FIFO command buffering), Stack (LIFO operation logging) | Java |

HW2 was a pen-and-paper mathematical assignment and is not included here.

More assignments will be added here as they are organized.

## Notes

- Each assignment folder includes a `README.md` describing the problem, the design approach, and how to build and run the code.
- Course materials, grading rubrics, and any instructor-provided files are intentionally excluded; only original implementation work is included.
